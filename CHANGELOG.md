# 📝 My Keep — История версий

## v4 — Исправления и улучшения

### ✅ Что исправлено

| Проблема | Решение |
|---|---|
| Нет кнопки «Вернуть из архива» | Добавлен проп `isArchiveView` → большая кнопка «Вернуть» |
| ColorPicker вертикальный | Фиксированная ширина `280px` + `flexShrink: 0` |
| Удаление папки ломало заметки | `writeBatch` переносит заметки в корень |
| Только локальная загрузка фото | Два таба: «С устройства» + «По ссылке» |

### 🔧 Изменения в файлах

**ColorPicker.jsx**
- Горизонтальная прокрутка вместо сетки
- Галочка на выбранном цвете
- Фиксированная ширина 280px

**NoteCard.jsx**
- Проп `isArchiveView` для режима архива
- В архиве: кнопка «Вернуть» + «Удалить»
- Вне архива: полный набор действий

**useFolders.js**
- `deleteFolder` теперь использует `writeBatch`
- Заметки переносятся в корень перед удалением папки

**ImageUpload.jsx**
- Два таба: файл / URL
- Drag & drop для файлов
- Валидация URL изображений

**useNotes.js**
- Добавлена функция `uploadImageFromUrl`
